🌐

Coordinate Systems

World (Xw, Yw, Zw): The global 3D reference frame (+Z points Up).
Camera (Xc, Yc, Zc): Origin at the camera center, Z-axis points forward.
Image (u, v): 2D pixel coordinates on the image plane.

📐

Extrinsic Parameters

The matrix [R | t] performs a rigid body transformation. R is a 3×3 orthogonal rotation matrix (RTR = I, det(R) = 1). t is a 3×1 translation vector. Together as a 4×4 matrix, they map world coordinates to camera coordinates.

🔭

Intrinsic Parameters (Pinhole Model)

The matrix K encodes the camera's internal geometry:
fx, fy: Focal lengths in pixels.
cx, cy: Principal point (image center offset).
s: Skew coefficient (usually 0 for modern cameras).

📊

Homogeneous Coordinates

By adding a 4th component (w=1), we can represent both rotation and translation in a single 4×4 matrix multiplication. This unifies affine transformations and enables perspective division: (x, y, z, w) → (x/w, y/w, z/w).
